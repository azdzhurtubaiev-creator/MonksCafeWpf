\## Особливості



| # | Функція | Валідація |

|---|---------|-----------|

| 1 | Add Item | опис 3–20 символів, ціна > 0, максимум \*\*5\*\* позицій |

| 2 | Remove Item | номер у діапазоні, `0` — скасувати |

| 3 | Add Tip | відсоток \*(0–100)\*, фіксована сума або без чайових |

| 4 | Display Bill | Net, Tip, GST (5%), Total у діалоговому вікні |

| 5 | Clear All | скидає позиції та чайові |

| 6 | Save to file | імʼя файлу 1–10 символів |

| 7 | Load from file | відновлює позиції \*\*та\*\* чайові |

| 8 | Exit | вихід |



\*\*Застосунок ніколи не аварійно не завершується\*\* на некоректному введенні — усі помилки показуються через `MessageBox`.



\## Архітектура



\- `BillService.cs` — бізнес-логіка, \*незалежна від типу застосунку\* (переносна з консольної версії)

\- `MainWindow.xaml` — інтерфейс: темна тема, помаранчеві кнопки, `DataGrid` позицій

\- `MainWindow.xaml.cs` — обробники подій, кожен ≤ 10 рядків
